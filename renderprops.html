Render Props

The term "render prop" refers to a technique for sharing code between React components using a props whose value in a function.

Step-by-Step
1. create a component where we code the common fuctionallity between components
2. pass the arrow function as props to this component
3. Arrow fuction accept the common fuctionallity from this component and pass though to components(need common fuctionallity).
4. the new component only return the props with fuctionallity as arguments.

Example: 

        App.js
            function App(){
                return (<>
                <CounterProps>
                {(count,IncrementCount)=><ClickCounterProps count={count} IncrementCount={IncrementCount}}</ClickCounterProps>}/>
                <CounterProps>
                {(count,IncrementCount)=><HoverCounterProps count={count} IncrementCount={IncrementCount}}</HoverCounterProps>}/>
                </>)
            }
            
            CounterProps.js
            class CounterProp extends Component {
                constructor(props) {
                    super(props)
            
                    this.state = {
                        count: 0
                    }
                }
                IncrementCount = () => {
                    this.setState(prevState => ({
                        count: prevState.count + 1
                    }
                    ))
                }
                render() {
                    return (
                        <div>
                            {this.props.children(this.state.count,this.IncrementCount)}
                        </div>
                    )
                }
            }

            ClickCounterProps.js
            class ClickCounterProp extends Component {
                render() {
                    return (
                        <button onClick={this.props.IncrementCount}>Click {this.props.count} times-props</button>
                    )
                }
            }
