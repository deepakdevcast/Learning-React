Introduction to React

developed and mantained by Facebook

What is React?

React is an Open Source JavaScript Library(not framwork :) ) and main focus to maintain and boost up the UI(User Interface).

We can used React for routing and HTTP request handling means React can used to build a full-fledged web appication.

React follow Component-base Architecture

React is declarative in nature means just declare what you want and the React take care of rest

DOM updates gracefully handled by React

Prerequisites

HTML, CSS and JavaScript(ES6 would be advantages)

JavaScript: map, reduce, filter and 'this' keyword

ES6: let and const, arrow function, object literal, default parameter template literals , rest and spreas operation, async, destruting assignment.
