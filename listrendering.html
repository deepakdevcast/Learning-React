List Rendering - React

using to render multples object at once such as course list, name list ...

Step-by-Step

1. get all data in array/list

2. create a new varibles to store all JSX of each element

3. optional if you have data as object then create new js file to handle the one JSX and then use to populate the new variables(step 2)

4. return the new varible

5. if you follow all the above step it will work perfectly fine but it's give a warning to add key

key is a special attribute when rendring list, the key has to assign some value which has to be unique for every element in list,
key is not accessble by clild component and key helps in efficient updates

you can use index as a key if
1. elements in list not have a unique id and
2. list is static and will not chage and
3. list will never be reordered or filtered

example: (best practice to follow step 3 in case of object) 
        function CouseRender() {
            let courses = [
                {
                id: 1,
                name: "HTML CSS Fundamental",
                price: "20$",
                rate: "4.5/5"
                },
                {
                    id:2,
                    name: "JavaScript Advanced",
                    price: "25$",
                    rate: "4.6/5"
                },
                {
                    id: 3,
                    name: "React AtoZ",
                    price: "20$",
                    rate: "5/5"
                },
            ]
            const courseList = courses.map(course=><Course key={course.id} course = {course}/>)
        return (
            <div>
                {courseList}
            </div>
        )
        }
        
        function Course({course}) {
            return (
                <h3>{course.name} price: {course.price} rate: {course.rate}</h3>
            )
        }
