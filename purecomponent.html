React Pure Component

Pure Component is second way to create Class Component.

example 

class A extends PureComponent{
        render(){
            return <h1>Pure Component<h1>
        }
    }

Advantage over simple Component - Pure Componet did not re-rendered if there is no difference because Pure Component implements shouldComponentUpdate method with shallow props and state comparison(preformance boostup)

Note: although Pure Component is only for Class Component but you can achieve this also in Functional Component using "React memo"

Functional Component: React momo

For get same beheivior as Pure Component in Functional component. Memo Component is a higher order component
Achieved by just change "export defualt name" to "export default React.memo(name)"

example 
function B{
            return <h1>Pure Component<h1>
        export default React.memo(B)
        }
