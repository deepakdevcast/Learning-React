Event Binding

It is used to change state using event handler

React have 4 ways to bind the this keyworg

1. using bind method
2. call the handler using arrow function
3. defined the bind method in class constructor
4. create handler method using arrow fuction

example: 

        class ClassClick extends Component {
            constructor(props) {
                super(props)
            
                this.state = {
                    greet: "Wellcome"
                }
                this.clickHandler = this.clickHandler.bind(this)
            }
            
            clickHandler(){
                this.setState(prevState =>({
                    greet: "thank you"
                }),()=>{console.log(`${this.state.greet}`)})
                
            } 
            // 4th way
            arrowClickHandler = ()=>{
                this.setState(prevState =>({
                    greet: "thank you"
                }),()=>{console.log(`${this.state.greet}`)})
            }
            render() {
                return (
                    <div>
                        {/* 4 ways to bind and 3,4 is the best ways */}
                        
                        {/* 1. by using bind method  not a good way */}
                        {/* <button onClick={this.clickHandler.bind(this)}>Class Click</button> */}
                        
                        {/* 2. using arrow function not that good */}
                        {/* <button onClick={()=>this.clickHandler()}>Class Click</button> */}
        
                        {/* 3. defined the bind in the constuctor of class */}
                        {/* <button onClick={this.clickHandler}>Class Click</button> */}
        
                        {/* 4. create the handler function using arrow function */}
                        <button onClick = {this.arrowClickHandler}>Class Click</button>
                        </div>
                )
            }
        }
