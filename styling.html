Styling React Component

basic 4 ways to style react component

1. CSS Stylesheets
2. CSS Inline Syling
3. CSS Modules
4. CSS in JS Libaries(Styled Component)

1. CSS Stylesheets

i. create a css stylesheet for that component
ii. import this css file into that component
iii. and add class or id to tags
iv. (optional) pass class name as props to component to add

note: if want add multiple classes then use combination of template $ to add variables

2. Inline Style

i. In React, the syle attribute accept object as value in which all the syling define
ii. first, create a varible and assign all the styling you want(React follow Camelcase name pattern)()
iii. and then assign this varible to style attribute

3. CSS Modules

i. create a file with extension .module.css ex: styling.module.css
ii. their is different in importing simple css(import './filename.css') and module css(import style from './filename.module.css') iii. then assign a class from module ex: lt;h1 className={style.name}>module text<h2>

Important!!: advantage to use CSS module is that can be access without importing into child (local Scope)

Example:

        Component
        import './stylec.css'
        import styles from './style.module.css'
        const inlinestyle = {
            fontSize: '25px',
            color: 'red'
        }
        function Styling(props) {
            let primary = props.primary ? "purp":""
            return (
                <div>
                    {/* 1. StyleSheet */}
                    <h2 className={`${primary} xl`}>Stylesheet</h2>
                    {/* 2. Inline */}
                    <h2 style={inlinestyle}>Inline</h2>
                    {/* 3. Css Module */}
                    <h2 className={styles.warning}>module text</h2>
                    </div>
            )
        }

        css file(stylec.css)
        purp{
            color: purple;
        }
        .xl{
            font-size: 50px;
        }

        css module(style.module.css)
        .warning{
            color: yellow;
        }
