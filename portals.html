React Portals

you noticed that everything you do always be in the root(app) tag to break through this use react portals

Step-by-Step

1. add new div tag in index.html file with id
2. create a component and do what you want before returning the JSX use ReactDOM.createPortal(<JSX>,document.getElementById("ids"))
3. after this add this component in App(main) component to render

Example: 

        Portal component

        import React from 'react'
        import ReactDOM from 'react-dom'
        function PortalComp() {
            return ReactDOM.createPortal(
                <>
                    <h2>React Portal</h2>
                </>,document.getElementById('react-portal')
            )
        }
        export default PortalComp

        parent Component

        fuction App(){
            return (
                <PortalComp/>
            )
        }
Important!! the Portal component is a child of App component although it's not a child of root in DOM means you can pass anything to parent component from portals
Best place is to use Portals is when using Modal Component or hovering something above and do changes in back
