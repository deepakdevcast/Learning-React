React Fragments

used to replace the root <div> tag in return

you must noticed than if you wanna return the multiple elements then you have to insert all element in div
the problem with this approch is this div tag also get rendered in DOM and sometime console gives warning.
the best appoach is to replace this div to React.Fragment .

example: 

    return (<React.Fragment>
            <h1>text&/h1>
            <p>para</p>
            </React.Fragment>)

Point- React.Fragment also accept key attribute it means it can be used in List Rendering

sortcut for using Fragment is to- replace <React.Fragment> to empty brakets <> (one limition of this sortcut is you can't pass key attribute)

example: 

    return (<>
            <h1>text&/h1>
            <p>para</p>
            </>)
