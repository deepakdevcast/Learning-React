Conditional Rendering

means rendering some html conditionally

In JS, 4 ways to do conditional rendering

1. if/else condition
2. Element variables
3. ternary operator(advantage: it can be used in the JSX/HTML)
4. Sort circuit operator (this working only in special case if you want to something or nothing)

example:
