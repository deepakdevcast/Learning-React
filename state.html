Component State

state is nothing but a private object maintained by Components

state is managed within the component

Variables declared in the function body

state can be changed


useState Hook - Function Components

this.state - Class Components

Step-by-Step

1. Create a Class Component

2. add constuctor and create this.state object with the properties

3. make a user-defined function to change state using this.setState(to replace old state to new object)

Example -

class Message extends Component {
        constructor() {
            super()
            this.state = {
                message: "Message to Readers"
            }
        }
        changeMessage(){
            this.setState({
                message: "Thank you"
            })
        }
        render() {
            return (
                <div>
                    <h1>{this.state.message}</h1>
                    <button onClick={()=>this.changeMessage()}>Click to see</button>
                </div>
            );
        }
    }

point:
1. if you try to change state without using setState then UI won't update but you can see change in console
2. this.setState in async function. you can run something after this by using Callback function
example: 

this.setState({
        count: this.state.count+1
    }, ()=>{
        console.log("Callback: ", this.state.count)
    })

3. when you have to update state based on the previous sate value, pass in a fuction as an argument instead of the regular object(Good way to code)
example:
this.setState(preState => ({
        count: preState.count + 1

    }), () => {
        console.log("Callback: ", this.state.count)
    })
