Event Handling

all the react event handler name follows camelcase example: onClick

it's except the object example: onClick={eventHandlerMethod}(notice function doesn't have brackets'()' )

fuctional Components

        function A(){
            function clickHandler(){
                console.log("clicked")
            }
            return (
                <div>
                <button onClick={clickHandler}></button>
                </div>
            )
        }

Class Components

        class ClassClick extends Component {
            clickHandler(){
                console.log("class button clicked")
            } 
            render() {
                return (
                    <div>
                        <button onClick={this.clickHandler}>Class Click<button>
                    <div>
                )
            }
        }

Important!! to use event handler to change state you have to use Event binding
